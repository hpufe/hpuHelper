## 加入我们

- **vue**  https://github.com/jeneser/vue

- **vux**  https://github.com/airyland/vux
- **cordova** http://cordova.apache.org/

## 如何开始

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
